# 宿析OS酒店语音热词表

只有上下文支持时才纠正，不做机械全局替换。

## 项目与平台词

| 标准词 | 可能的语音转录形式 | 判断条件 |
| --- | --- | --- |
| 宿析OS / SUXIOS | 熟悉OS、宿系OS、速析OS | 当前项目、酒店系统或宿析OS功能语境 |
| 语音输入 | 随行输入、随形输入、随机输入 | 正在讨论口述、麦克风或转录质量 |
| Codex | CodeX、扣代克斯、代码助手 | 正在讨论当前AI助手、Skill或插件 |
| 携程 | 协程、写程 | 酒店、OTA、商家后台、订单或流量语境；代码并发语境保留“协程” |
| 美团 | 每团、美团网 | 酒店、OTA、eBooking、订单或榜单语境 |
| OTA | O T A、欧提诶 | 渠道、平台、订单、流量或收益语境 |
| PMS | P M S、皮埃姆艾斯 | 酒店管理系统、房态、入住或日报语境 |
| CRS | C R S | 中央预订、渠道或房态价量语境 |
| eBooking | E Booking、易booking | 携程/美团商家后台语境 |
| Profile | profile、浏览器档案 | 登录态、浏览器会话或门店隔离语境 |
| Cookie | cookie、曲奇 | 授权会话语境；按敏感凭证处理 |
| Payload | 配load、请求体 | 接口、采集或响应证据语境 |

## 酒店经营与收益词

| 标准词 | 可能的误转 | 说明 |
| --- | --- | --- |
| 收益分析 / 收益管理 | 受益分析、受益管理 | 酒店收入、房价、利润或调价语境 |
| ADR / 平均房价 | A D R | 必须明确 OTA 渠道还是全酒店口径 |
| OCC / 入住率 / 出租率 | O C C、入驻率 | 分母和数据范围不清时不能自行补值 |
| RevPAR | rev par、瑞帕 | 每间可售房收入；渠道或全酒店口径需标明 |
| Net RevPAR | net rev par | 需要净收入与可售房晚口径对齐 |
| GOP | G O P | 酒店经营毛利润语境 |
| ROI / IRR | R O I、I R R | 投资回报与内部收益率语境 |
| ROAS | R O A S | OTA广告归因收入与花费语境 |
| 间夜 | 间页、建业 | 订单、入住、渠道成交量语境 |
| 可售房晚 | 可售房、可售房间 | 计算 OCC/RevPAR 时保留时间粒度 |
| 房态 | 房贷、房台 | 库存、可售、满房或关房语境 |
| 竞品 / 竞对 | 精品、镜品 | 酒店竞争圈、价格比较或排名语境 |
| 调价 | 条价、调节 | 房价策略、收益建议语境 |
| 投决 | 投绝、投诀 | 投资决策、项目测算语境 |

## 建议加入输入法个人词库的热词

宿析OS、SUXIOS、携程、美团、OTA、PMS、CRS、eBooking、Profile、Payload、ADR、OCC、RevPAR、Net RevPAR、GOP、ROI、IRR、ROAS、间夜、可售房晚、房态、房价、库存、收益管理、竞品、竞对、转化率、曝光、佣金率、调价、数据健康、平台酒店ID、系统酒店ID、OTA渠道口径、全酒店口径。

## 业务链固定表达

`携程/美团 OTA 数据 -> 收益分析 -> AI 决策 -> 运营管理 -> 投资决策`

任何语音纠错都不能破坏这条链中的来源、门店、日期、数据范围和验证状态。
